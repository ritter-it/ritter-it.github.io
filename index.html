OGConsole

An interactive debugging toolkit and canvas inspector for OmniGraffle Automation scripting.

↓ View on GitHub ↓ Download ZIP

Overview

OGConsole is a plug-in for OmniGraffle that displays a structured inventory of the current canvas - shapes, layers, IDs, properties — and registers shorthand commands callable directly from the console without parentheses.

It is designed to make interactive debugging and scripting exploration as fast and convenient as possible.

Installation

Download the .omnijs file(s) from the repository

Double-click to install into OmniGraffle

The plug-in appears under Automation → OGConsole

Open the console via Automation → Automation Console and run the plug-in

Requirements
OmniGraffle 7.4 or later (macOS) · Omni Automation enabled

Console Commands

hShow help overview
iCanvas inventory (short overview)
igGraphic inventory (full shape details)
idDocument inventory
shapesList all shapes on current canvas
shape(n)Show all properties of shape at index n
shapeProps(n)Show all properties incl. prototype chain
selShow currently selected shapes
findByName(str)Find shapes whose name contains str
findByText(str)Find shapes whose text contains str
json(n)Copy shape n as JSON to clipboard
cnvsReference to the current canvas
cClear the console
Note
Commands without parentheses are implemented as JavaScript property getters - no parentheses needed.

Console Getter Pattern

OGConsole registers functions as property getters on globalThis - this enables calling them without parentheses directly in the console.

// register a getter on globalThis Object.defineProperty(globalThis, 'i', { configurable: true, get: () => dumpCanvas(cnvs()) }) // call without parentheses in the console i
Important
configurable: true is required to allow re-registration when the plug-in is executed multiple times in the same session.
